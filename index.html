Become a Node

Follow these steps to set up your own NEBEK node and start earning rewards

1

Prepare Your Hardware

Get your Raspberry Pi ready to join the NEBEK network:

  • Download Raspberry Pi OS Lite (64-bit)
  • Flash the OS to your MicroSD card
  • Create an 'ssh' file in the boot partition
  • Insert the card and power on
2

Basic Configuration

Connect to your Raspberry Pi via SSH:

ssh pi@raspberrypi.local

Update your system:

sudo apt update && sudo apt upgrade -y

Install required packages:

sudo apt install -y wireguard wireguard-tools curl jq
3

Install NEBEK Node

Install the node software with one command:

curl -sSL https://get.nebek.network | bash

Or clone from GitHub and build manually:

git clone https://github.com/nebek/node-client cd node-client cargo build --release sudo cp target/release/nebek-node /usr/local/bin/
4

Generate Your Keys

Create your node identity and secure keys:

nebek-node keygen

Create a configuration file at /etc/nebek/config.json:

{ "node_id": "YOUR_NODE_ID", "private_key": "YOUR_PRIVATE_KEY", "public_key": "YOUR_PUBLIC_KEY", "blockchain_endpoint": "https://blockchain.nebek.network", "bootstrap_nodes": ["bootstrap1.nebek.network", "bootstrap2.nebek.network"], "wireguard_interface": "wg0", "max_clients": 20, "ip_rotation_enabled": true }

Hardware Requirements

Run a NEBEK node with minimal hardware requirements

Recommended Hardware

  • Raspberry Pi 4 (2GB RAM or better)
  • MicroSD card (16GB+)
  • Power supply
  • Ethernet connection (recommended)
  • Optional: Case with cooling fan

Network Requirements

  • Stable internet connection
  • Minimum 10 Mbps upload speed
  • Minimum 20 Mbps download speed
  • Open UDP port 51820
  • Dynamic IP or ability to rotate IP

Advanced Setup

  • Enable IP forwarding
  • Setup as a systemd service
  • Configure IP rotation methods
  • Multiple interface support
  • Performance monitoring tools

Security Considerations

  • Use a dedicated Raspberry Pi
  • Consider using a separate network
  • Keep node software updated
  • Never share your private keys
  • Regularly check for suspicious activities
